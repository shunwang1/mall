The newbee-mall project is a complete e-commerce system including the newbee-mall front-end and newbee-mall-admin back-end management system, developed based on Spring Boot 2.X and related tech stack. The front-end includes modules such as homepage, product category, new arrivals, homepage carousel, product recommendation, search, product details, shopping cart, order checkout, order process, user order management, member center, and help center. The admin system includes modules such as data dashboard, carousel management, product management, order management, user management, category management, and settings.
